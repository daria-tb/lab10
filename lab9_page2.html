Вступ

Цей звіт документує практичні навички та досвід роботи з мовою програмування Python. Метою проходження даної практики було опанувати основні функціональні можливості цієї мови праграмування, а завдання включали побудову графіків функцій, роботу з матрицями, векторами, системами рівнянь та іншими видами завдань, що повʼязані з лінійною алгеброю та аналітичною геометрією. Для виконання всіх цих завдань була використана стандартна бібліотека Python та її модулі, а також бібліотеки Matplotlib, NumPy, SimPy. У цьому звіті буде представлено виконані завдання та досягнуті результати під час практики.

Python і його функціональні можливості

Мова програмування Python відома своєю простотою та читабельність коду, що сприяє швидкому розробленню програм. Мова має чіткі конструкції для роботи з функціями, що дозволяє створювати код, який легко розуміти та змінювати. Python використовується в різноманітних сферах, таких як створення ігор та веб-застосунків, розробка внутрішніх інструментів для різноманітних проєктів. Ця мова програмування також широко застосовується в наукових дослідженнях та розв'язанні математичних завдань. Однією з важливих переваг Python є його постійні оновлення та доповнення, що дозволяють використовувати передові технології та інновації в галузі штучного інтелекту, машинного навчання та інших сучасних рішень. Також, Python приваблює своєю легкістю в навчанні. На просторах інтернету яснують різні онлайн-ресурси, курси та документації, які роблять його доступним для новачків у програмуванні.

Значення функцій в заданих точках

Для цих завдань зазвичай використовують модуль math з основної бібліотеки Python. Цей модуль надає функції для виконання математичних операцій та обробки числових даних:

  1. Основні математичні функції (та тригонометричні): косинус, синус, тангенс, експоненціальна функція, логарифми та інші.
  2. Константи: такі як π (pi) та е (e/exp(x)).
  3. Операції з числами: функції для округлення чисел, отримання абсолютного значення, піднесення до ступеня та інші операції.
  4. Функції для роботи з числовими даними: переведення числа в цілі або з плаваючою комою, отримання цілої частини числа та інші.

У цьому розділі представлені завдання на визначення значень функцій f(x) у різних точках.

Побудова графіків функцій

Цей розділ повʼязаний з подубовою графіків. Мова програмування Python надає багато зручних інструментів для візуалізації даних, а найпопулярніший з всіх - matplotlib.pyplot. Цей модуль дозволяє створювати графіки, діаграми та інші види візуалізацій з легкістю. Модуль matplotlib.pyplot дозволяє створювати різні типи графіків, такі як лінійні графіки, стовбчасті діаграми, колові діаграми, гістограми та інші. Також він дозволяє налаштовувати заголовок, мітки осей, легенди, змінювати колір та стиль ліній. А простий інтерфейс робить цей модуль дуже популярним серед розробників.

У цьому розділі виконані такі завдання:

  1. Лінійні графіки
  2. Степеневі графіки
  3. Показникові графіки
  4. Графіки композицій функцій
  5. Логарифмічні графіки
  6. Тригонометричні та обернені тригонометричні графіки
  7. Гіперболічні графіки
  8. Графіки в полярній системі координат

Перетворення алгебраїчних виразів

Перетворення алгебраїчних виразів у Python виконують за допомогою бібліотеки SymPy. Ця бібліотека надає можливості для роботи з символьними виразами, розв'язування алгебраїчних рівнянь, диференціювання та інтегрування функцій та вирішення систем рівнянь. SymPy необхідна для вирішення завдань у математичній фізиці, інженерії і не тільки.

Основні функції:

  1. Розв'язання рівнянь, які містять символьні змінні (тобто ми замість числових значень, працюємо з алгебраїчними виразами, які можна змінювати). Це надає можливість розв'язувати лінійні та нелінійні рівняння.
  2. Операції диференціювання та інтегрування для символьних функцій.
  3. Робота з матрицями, лінійною алгеброю.
  4. Може створювати графіки (обʼєднуючись з бібліотекою matplotlib).

У цьому розділі представлені такі завдання як:

  • Розкладення раціонального дробу на прості.
  • Ділення многочлена. Знаходження частки та залишок від ділення.
  • Розкладення многочлена по степені.

Матриці

Більша частина роботи з матрицями виконується за допомогою модулю NumPy. Використання NumPy для операцій з матрицями робить обчислення більш читабельними. Ось деякі основні аспекти роботи з матрицями:

  1. Створення матриці за допомогою функції np.array(), np.zeros(), np.ones().
  2. Операції з матрицями (додавання, віднімання, множення, транспонування).
  3. Функції для виконання розв'язування систем лінійних рівнянь, знаходження оберненої матриці та власних значень.
  4. Визначення детермінантів (determinant = np.linalg.det(matrix)).

Такі завдання, представлені у практичній частині цього розділу:

  • Обчислення добутку матриць.
  • Обчислення детермінанту.
  • Обчислення оберненої матриці.
  • Визначення рангу матриці.
  • Транспонування матриці.

Висновки

Робота з мовою програмування Python виявилась дуже корисною, легкою та плідною. Легкою вона була тому, що Python дуже гнучка мова, має легкий синтаксис - саме те, що потрібно для початківців. Наявність великого вибору бібліотек та модулів значно полегшила виконання завдань та їх читабельність. А саме розширений вибір бібліотек для виконання математичних завдань. Бібліотеки NumPy, SymPy, Matplotlib надають велику кількість інструментів для роботи з числовими даними, символьною математикою, алгоритмами та різноманітними графіками функцій, що значно полегшує написання коду. Саме тому вона стає важливим інструментом для студентів, вчених і звісно для програмістів. Таким чином, робота з Python виявилася надзвичайно продуктивною. А те, що написання коду виявилось дуже простим, зробило проходження практики приємний заняттям.
